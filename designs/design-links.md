## Notas sobre Diseños Exportados

Si prefieres subir los archivos de diseño directamente al repositorio en lugar de mantener enlaces:

1. **Exporta desde tu herramienta**:
   - Google Stitch: Exporta como PNG/JPG
   - Figma: Exporta componentes o pantallas
   - Adobe XD: Exporta artboards

2. **Organiza en carpetas**:
   ```
   designs/
   ├── google-stitch/
   │   ├── wireframes/
   │   ├── mockups/
   │   └── prototypes/
   └── figma/
       └── exported/
   ```

3. **Sube usando Git**:
   ```bash
   git add designs/
   git commit -m "Agregar diseños exportados"
   git push
   ```

Ver [designs/README.md](../designs/README.md) para más detalles.
